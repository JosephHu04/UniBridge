# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - Initial Release

### Added
- 🏆 QS 2025 Top 100 global university rankings integration
- 👨‍🏫 Professor review system with 3-dimension radar chart rating (Character & Attitude, Academic Ability, Resources & Platform)
- 🏫 School review system with 10-dimension rating (Reputation, Location, Facilities, Opportunities, Safety, etc.)
- 🌍 Academic navigation tree: Region → School → Department → Major
- 📌 Professor bookmark/favorite system
- 👍 Upvote/downvote system for review quality control
- 🔐 User authentication (register, login, logout) with BCrypt session tokens
- 🛡️ ActiveAdmin admin panel for content moderation
- 📡 RESTful JSON API (30+ endpoints)
- 🎨 React 17 + Redux 4 + Recharts frontend
- 🧪 RSpec test suite
- 📖 Bilingual README (中文 / English)
